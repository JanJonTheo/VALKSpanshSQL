# Spansh Project Integrated

## Start GUI

```bash
python -m spansh_project_integrated.gui
```

## Enthalten

- Import-Tab nur für Spansh-Import
- Planner-/Abfrage-Tab mit Einzel- und Mehrfach-Referenzsystemen
- Discord-Vorschau und Discord-Webhook-Versand
- CSV-Export
- Auto-Tuning aus GUI und CLI entfernt
- Poll-Intervall in der GUI entfernt, intern fest auf 10 Sekunden

## Hinweis

Die bestehende DuckDB-Struktur wird weiterverwendet. Es werden keine Schema-Anpassungen für das Planner-Tool vorgenommen.
